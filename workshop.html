Digital Signal Processing Workshop

Abstract:

  • Digital signal processing (DSP) is the use of digital processing, such as by computers or more specialized digital signal processors to perform a wide variety of signal processing operations.
  • The digital signals processed in this manner are a sequence of numbers that represent samples of a continuous variable in a domain such as time, space, or frequency.

Prerequisites:
  • No prerequisites required.

Topics to be covered:
  • Basics of the Topic.
  • Types of Signals.
  • Python Libraries- NumPy and Matplotlib.
  • Plotting of signals using Python in Google Collab.
  • Applications of DSP.
  • Real time building of voice changing filters.

Learning Outcomes:
  • In this workshop, we will learn about various signal manipulation algorithms and build projects in Python. We will also learn data visualization and analysis using various libraries in Python.

WeBots Workshop

Abstract:

  • Webots is a professional mobile robot simulation software package. It offers a rapid prototyping environment, that allows the user to create 3D virtual worlds.
  • User can program each robot individually to exhibit the desired behavior. Webots contains a large number of robot models and controller program examples to help users get started.

Prerequisites:
  • A basic knowledge of the C, C++, Java, Python or MATLAB programming language is necessary to program your own robot controllers.
  • Webots Software should be installed in everyone’s computer to do hand’s on practice.

Topics Covered:
  • Introduction to basic electronics,
  • Basic introduction to robots and robotics,
  • Introduction to Webots software and E-puck mobile robot
  • Instructions to simulate a robot in Webots

Learning Outcomes:
  • In this workshop, we will learn about Webots software and how to use it to simulate a robot, create environment and program the robots accordingly. This will help us to learn about Robotics, electronics, simulation of robots and to explore newer technologies.
